The Frontend Developer Career Path Course by Scrimba

When I first started this course in October 2021, I was instantly impressed by their scrim technology. It allows you to pause the video, edit, save the code displayed on the screen, and then resume the tutorial. This interactive mode of learning supersedes, by far, all the other platforms and the structured course module is exactly what I was looking for. I've built over 10 real-world projects thus far and Scrimba has helped me develop my skills in JavaScript and React more confidently.

Initially my aim was to obtain the completion certificate within 4 months. However, I decided to put my progress on hold and focused on building this portfolio website instead to jumpstart my career in the tech industry.
